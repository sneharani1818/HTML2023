Tag Name

  • Doctype: defines that this document is an HTML5 document.
  • html: root element of an HTML page.
  • head: contains meta information about the HTML page.
  • title: specifies title of page, which is shown as tab heading.
  • body: defines body of the element, and contains all the visible elements.
  • ul: unordered list
  • li: list element
  • br: break line/ shift contents to next line
  • b: bold
  • i: italics
  • img: image tag with src attribute that specifies the image address.
  • a: anchore tag that specifies any link with attribute href that specifies the link to that external webpage.
  • ol: ordered list with numbering
  • p: add some white space (a margin) before and after a paragraph
  • hr: horizontal rule/line
  • pre: preserves both tab spaces and line spaces
  • link: to link external files, like favicon, css files etc. We have to define type of file, it's href(hyper link)
  • blockquote: define a section that is quoted from another website
  • q: to display text within quotations
  • abbr: to display abbreviations such as html, who, etc.
  • address: automatically display in italics
  • cite: to give title to pictures

  • bdo: to override current text direction
